# Frontend Engineering Skill
Project: Athena - AI Sports Intelligence Platform

## Purpose

You are responsible for building the complete frontend of Athena.

Athena is NOT a betting application.

It is an AI-powered sports intelligence platform that consumes real-time sports data from TxLINE and presents explainable insights generated by multiple autonomous AI agents.

The UI should feel like a premium Bloomberg Terminal, Stripe Dashboard, Linear, Vercel, and Perplexity AI combined.

Everything should communicate trust, intelligence, precision, and speed.

Never build generic dashboards.

Every page should feel intentional and production ready.

---

# Technology

Always use:

- Next.js App Router
- TypeScript
- TailwindCSS
- shadcn/ui
- Framer Motion
- React Query (TanStack Query)
- Zustand (only for local UI state)
- React Hook Form
- Zod

- date-fns
